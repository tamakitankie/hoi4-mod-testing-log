## Quick Start Guide

### For First-Time Users:
1. **Download the entire repository** (click Code â†’ Download ZIP)
2. **Extract all files** to a folder (e.g., `C:\HOI4_Tools\`)
3. **Double-click `RUN_SCRIPTS.bat`**
4. Follow the menu options

### Individual Scripts:
- **`RUN_SCRIPTS.bat`** - Main menu (start here!)
- **`verify_install.bat`** - System check only
- **`hoi4_nuke.py`** - Cache cleaner (needs Python)

### Common Issues & Fixes:

**"File not found" error:**
- Ensure all files are in the SAME folder
- Don't move files individually
- Download the whole repository

**Python not installed:**
- Download from [python.org](https://python.org)
- During installation, CHECK "Add Python to PATH"
- Or use the PowerShell version

**OneDrive issues:**
- Move the folder out of OneDrive synced locations
- Use `C:\HOI4_Tools\` or `Desktop\HOI4_Tools\`
- Disable OneDrive sync for the tools folder
